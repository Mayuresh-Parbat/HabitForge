# HabitForge

AI-Powered Habit Building & Personal Growth Platform

HabitForge is a smart habit tracking and self-improvement platform designed to help users build consistency, improve discipline, and achieve long-term personal goals. The platform combines productivity systems, analytics, and AI-driven insights to create a modern personal development experience.

---

# Features

## Habit Tracking System
- Create and manage daily habits
- Track habit completion progress
- Maintain consistency streaks
- Set habit goals and reminders

## AI Habit Assistant
- AI-generated habit suggestions
- Personalized productivity insights
- Smart motivation system
- Daily growth recommendations

## Analytics Dashboard
- Habit completion analytics
- Weekly and monthly performance tracking
- Consistency score visualization
- Personal growth statistics

## Productivity Features
- Daily routine planner
- Focus session tracking
- Goal management
- Progress monitoring

## Authentication & Security
- Secure user authentication
- JWT-based authorization
- Protected routes
- Password encryption

## Modern UI/UX
- Responsive design
- Clean and minimal interface
- Dark and light mode support
- Smooth animations and transitions

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/habitforge.git
```

---

# Frontend Setup

## Navigate to frontend

```bash
cd frontend
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

---

# Backend Setup

## Navigate to backend

```bash
cd backend
```

## Install dependencies

```bash
npm install
```

## Start backend server

```bash
npm run dev
```

---

# Environment Variables

Create a `.env` file inside backend folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_api_key
```
